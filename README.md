# Artificial Vision Project using OpenCV

This project was developed using OpenCV and focuses on object recognition and spatial mapping techniques in the field of Artificial Vision.

The project is divided into three main phases:

## Phase 1 – Object Counting
Detection and counting of beans present in the image using image processing techniques.

## Phase 2 – Object Classification
Classification of detected objects into:
- Beans
- Chickpeas

using shape and feature analysis.

## Phase 3 – Spatial Mapping
Mapping the detected objects in the image space by identifying their positions and coordinates.

---

## Technologies Used
- Python
- OpenCV
- NumPy

## Project Features
- Image preprocessing
- Contour detection
- Object segmentation
